# Task-03: Decision Tree Classifier

This project builds a Decision Tree model to predict whether a customer
will purchase a product or service based on demographic and behavioral data.

## Dataset
Bank Marketing Dataset (UCI Machine Learning Repository)

## Steps Performed
- Data loading and preprocessing
- Label encoding of categorical variables
- Train-test split
- Decision Tree training
- Visualization of the decision tree

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the script:
   python src/decision_tree_classifier.py

## Output
- decision_tree.png (visual representation of the model)
