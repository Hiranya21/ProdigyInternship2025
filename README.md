# Task-01: Data Visualization

This project visualizes the distribution of countries across income groups
using a histogram.

## Dataset
World Bank population metadata dataset.

## Steps
1. Load the dataset using pandas
2. Count countries by income group
3. Create a histogram using matplotlib
4. Export data and chart

## Output
- histogram_data.csv
- income_group_histogram.png
